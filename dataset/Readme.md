You can find a variety of Claude datasets for different fine-tuning tasks on Hugging Face. Here are some of the most relevant collections, organized for quick reference.

### 🤖 Prominent Claude Distilled Models & Datasets
If you're interested in reasoning distillation, several high-quality datasets are widely used for fine-tuning models to mimic Claude's problem-solving and structured thinking capabilities.

| Dataset / Collection Name | Key Use Case / Description |
| :--- | :--- |
| `TeichAI/claude-sonnet-4.5-high-reasoning-250x` | High-reasoning dataset from Claude 4.5 Sonnet |
| `TeichAI/claude-sonnet-4.5-high-reasoning-700x` | Reasoning traces from Sonnet 4.5 (700 samples) |
| `nohurry/Opus-4.6-Reasoning-3000x-filtered` | Curated Opus 4.6 reasoning traces (3k+ samples) |
| `angrygiraffe/claude-opus-4.6-4.7-reasoning-8.7k` | Opus 4.6/4.7 reasoning dataset (8.7k samples) |

### 📝 Instruction-Tuning Datasets
For more general instruction-following tasks, these large-scale datasets are excellent foundations.

*   **`Sao10K/Claude-3-Opus-Instruct-15K`** – **15,000** instruction-response pairs from Claude 3 Opus across a wide variety of tasks.
*   **`kalomaze/Opus_Instruct_25k`** – A large collection of **25,000** entries, often used as a base for further fine-tuning.
*   **`rahmanazhar/claude-3.7-sonnet-reasoning`** – JSONL dataset where responses include `<think>` tags for explicit step-by-step reasoning.

### 💻 Code & Writing Datasets
Claude is also widely used to generate specialized content. Check out these domain-specific datasets:

*   **Code Generation**: `Data-Agora/code_claude3.5_sonnet_10000` – **10,000** code generation examples from Claude 3.5 Sonnet.
*   **Creative Writing**: `Gryphe/Opus-WritingPrompts` – **3,008** short stories generated by Claude Opus.
*   **Creative Writing Instruct**: `Nopm/Opus_WritingStruct` – Over **6,000** creative writing instructions cleaned by Anthropic.

### 🧑‍⚖️ Preference (DPO) Datasets
If you're working with **Direct Preference Optimization (DPO)** for alignment, consider these:

*   `nbeerbower/Arkhaios-DPO`: A preference dataset aimed at reducing archaic language in LLMs.
*   `mlfoundations-dev/SCP_40k-claude-3-7-sonnet-16k`: Structured solutions with reasoning trajectories, useful as a comparison base.

### 💻 How to Download and Use These Datasets
All datasets on Hugging Face can be explored and downloaded directly. Here's how:

1.  **Visit the Dataset Page**
    Simply copy the full Hugging Face dataset ID (e.g., `Sao10K/Claude-3-Opus-Instruct-15K`) into your browser.

2.  **Load with the `datasets` Library**
    ```python
    from datasets import load_dataset

    # Load a dataset directly into memory for fine-tuning
    dataset = load_dataset("Sao10K/Claude-3-Opus-Instruct-15K", split="train")
    print(dataset[0])
    ```

3.  **Direct Download for Manual Use**
    Most dataset repositories also offer a "Download All" option on the "Files" tab, providing a direct link to download the raw files for local inspection or processing.

If you're interested in a specific use case or want to contribute a new dataset, the Hugging Face Hub has extensive documentation to get you started.
